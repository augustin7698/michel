Comment ça marche ?

Si vous savez coder en javascript (même si vous êtes un débutant) vous pouvez aider au développement de Michel, pour cela, clonez le projet sur votre machine puis proposer les modifications grâce à GitHub.


Si lorque vous entendez javascript vos oreilles saigne, vous pouvez toujour proposer des idées de nouvels fonctionnalités par Discord ou GitHub.

Comment coder ?

Si vous êtes partants pour rejoindre cet formidable aventure de développpement, vous devrez coder une boucle if qui, lorsque le message (variable input) contient x message éxécute une fonction personnalisé éxemple:

if (input.includes("frite") || input.includes("pizza")) {
    send("je sens que vous avez faim");
}


output = getkeyword(input) permet de supprimer les caractères spéciaux et obtenir le mot clé d'une phrase courte.
