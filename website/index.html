Comment ça marche ?

Un alghorythme javascript analyse le text que vous donnez à Michel, celui-ci cherche des mots clé dans votre text, si aucun ne correspond, il demande au serveur de lui donner les recommandations de la communautées, chacun peut apporter une idée de nouvelle fonctionnalité.

Comment coder ?

Pas besoins de savoir coder pour implémanter de nouvelles fonctions, vous pouvez le faire via l'onglet améliorer Michel. Mais si vous avez des idées plus avancé vous pouvez les proposer par GitHub, pour cela, le controle au moins partielle de javascript est requit.
